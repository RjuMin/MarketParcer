
Получение данных:
Клиент отправляет POST запрос на /api/parse с URL товара
Сервис парсера загружает страницу через Guzzle
Использует DomCrawler для извлечения данных

Извлечение данных:
Цена: ищет элемент с data-auto="mainPrice"
Название: извлекает текст из тега h1
Рейтинг: ищет элемент с data-auto="rating"

Возможные проблемы:

Яндекс может блокировать запросы без правильных заголовков
Структура страницы может меняться
Ограничение на частоту запросов
